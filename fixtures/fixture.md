# Modern Cobalt2

> A modern take on the Cobalt2 colour scheme — deep navy blues with muted accents.

## Features

- Flat `#062335` background across all surfaces
- Muted steel-blue highlights — no neon
- Palenight syntax colours on a Cobalt2 canvas
- Minimal border contrast (`#324858`)

## Installation

```sh
code --install-extension kud.modern-cobalt2-vscode
```

## Palette

| Role       | Hex       |
| ---------- | --------- |
| Background | `#062335` |
| Surface    | `#142e3f` |
| Border     | `#324858` |
| Highlight  | `#1d3546` |
| Text       | `#d6dadd` |
| Muted      | `#abb5c7` |

## Usage

Add to your `settings.json`:

```json
{
  "workbench.colorTheme": "Modern Cobalt2",
  "editor.fontFamily": "'JetBrains Mono', monospace",
  "editor.fontLigatures": true
}
```

## Inspiration

Based on [OpenCode Desktop](https://opencode.ai)'s Cobalt2 theme, with
**Palenight Operator** syntax colours and a fully flat VS Code workbench.
